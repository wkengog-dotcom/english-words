英単語ニュアンス図鑑

中学2年生向けに、似ている英単語の違いをイラストで楽しく確認できます。カードや表の「イラストを見る」を押すと、大きな画像が表示されます。

基本

eat / have

「食べる」の違い

eat は食べる動作、have は食事全体を表します。

重要

look / watch / see

「見る」の違い

look は意識して見る、watch は動くものを見る、see は自然に見える。

重要

speak / talk / say / tell

「話す・言う」の違い

speak は言語、talk は会話、say は内容、tell は人に伝える。

基本

house / home

「家」の違い

house は建物、home は自分の居場所や心が落ち着く場所。

重要

borrow / lend

「借りる・貸す」の違い

borrow は相手から借りる、lend は相手に貸す。

発展

trip / travel / journey

「旅行・旅」の違い

trip は短い旅行、travel は旅行全般、journey は道のりや旅の過程。

発展

big / large

「大きい」の違い

big は日常的な大きさ、large は広さや規模の大きさ。

基本

like / love

「好き」の違い

like はふつうの好き、love はとても強い好き。

発展

start / begin

「始める」の違い

start は動き出す感じ、begin は物事の最初を始める感じ。

重要

job / work

「仕事」の違い

job は職業や役割、work は働くことや作業全体。

発展

clever / smart

「頭がいい」の違い

clever は機転がきく、smart は知識や理解力が高い。
